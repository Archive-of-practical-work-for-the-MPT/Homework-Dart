## 1.1.0

- Implemented full Battleship game with console interface
- Added Ship model with name and size attributes
- Created Board class for game grid management
- Implemented Player class for ship placement and attacks
- Added two game modes: Player vs Player and Player vs Bot
- Added customizable board sizes (8x8, 10x10, 12x12)
- Implemented ship placement validation
- Added visual board display with hit/miss indicators
- Added player names support
- Implemented automatic ship placement for bot
- Added unit tests for core functionality
