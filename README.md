# Практическая работа 7 - Галерея + музыка

## Описание

Как сделать чтобы работало?
- Скачайте расширение Dart и Flutter в VSC
- Запустите через `F5` в удобном вам эмуляторе

Возможны сбои в Url на некоторых устройствах - причина не ясна.

## Цель

Разработать приложение, которое работает с url и позволяет загружать фотографии, видео и аудио-музыку. Все данные должны сохраняться в базе данных.

## Задание

1. Реализовать галлерею, но нужно добавить возможность перемотки видео, а также чтобы просмотр всех имеющихся видео и фотографий был в виде ленты (как в инстаграме).

2. Реализовать раздел "музыка" в приложении и добавить туда возможность загрузки/добавления URL ссылки на аудио, также возможность его слушать, ставить на паузу и перематывать.

3. Сохранять все эти данные в базе данных Hive/SQLite, также использовать пакет location для добавления геопозиции при сохранении фотографии.
