# Практическая работа 7 - Работа с сетью

## Описание

Как сделать чтобы работало?
- Скачайте расширение Dart и Flutter в VSC
- Запустите через `F5` в удобном вам эмуляторе

## Цель

Разработать приложение, которое работает с сетью с использованием пакета Dio. Добавить класс interceptor, подключить API через веб-сайт api-ninjas. Реализовать функциональность.

## Задание

1. Разработать приложение, которое работает с сетью. Для этого использовать пакет Dio.
   
2. Добавить класс interceptor, который автоматически добавляет токен в заголовок, и использовать interceptor от пакета pretty_dio_logger.
   
3. Для работы с сетью использовать пакет Dio. создать класс interceptor, ответственный за добавление токена в заголовок, а также использовать interceptor от пакета pretty_dio_logger. для управления зависимостями использовать пакет get_it.
   
4. Для взаимодействия с API использовать веб-сайт https://api-ninjas.com. если у вас есть собственное API, допускается его использование.
  
5. Приложение должно обладать функциональностью (не две кнопки)
