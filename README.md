# Практическая работа 5 - Заметки

## Описание

Как сделать чтобы работало?
- Скачайте расширение Dart и Flutter в VSC
- Запустите через `F5` в удобном вам эмуляторе

## Цель

Реализовать приложение заметки на Flutter

## Задание

1. Создать приложение на Flutter, которое будет представлять собой заметки. 
2. Пользователь сможет вводить текст в поле, и при нажатии на кнопку "Сохранить" введённый текст будет отображаться ниже в виде списка.
3. Добавьте возможность удаления заметок из списка.
4. Добавьте возможность редактирования существующих заметок.
