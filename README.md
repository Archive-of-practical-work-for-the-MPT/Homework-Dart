# Практическая работа 2 - Морской бой

## Описание

Как сделать чтобы работало?
- Скачайте расширение Dart в VSC
- Запустите код через терминал командой `durt run`

## Цель

1. Создайте консольное приложение "морской бой"
2. Игра должна позволять двум игрокам по очереди делать ходы, размещая свои корабли и атакуя корабли противника
3. Необходимо реализовать модель Ship, которая будет содержать параметры (например, название, размер и т.д.), (создайте атрибуты, которые вам необходимы для полноценной реализации программы, также можете реализовать другие классы при их необходимости).
4. Реализовать игру друг против друг (сначала игроки размещают корабли на поле, потом по очереди ходят, после каждого хода должно выводиться состояние поля своё и соперника)
5. Перед сменой хода игра должна указать, что теперь ходит другой игрок. во избежание подглядывания, после начала хода другого игрока, консоль должна очиститься
6. Реализовать возможность добавления и отображения в игре имени игрока, который сейчас делает ход
7. Реализовать возможность изменения размера поля (от изменения размера поля, должно меняться и кол-во кораблей), (можно делать фиксированное изменение размера поля, для того, чтобы избежать нарушения баланса в игре, например: "выберите размер поля 10x10, 14x14 и т.д." не менее 3-х вариаций таких полей)
8. Реализовать игру против бота. при игре в данном режиме, человек не должен видеть состояние поля бота ни при каком условии (имеется ввиду расположение его кораблей)
