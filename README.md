# Практическая работа 6 - Мини кинопоиск

## Описание

Как сделать чтобы работало?
- Скачайте расширение Dart и Flutter в VSC
- Запустите через `F5` в удобном вам эмуляторе

## Цель

Реализовать приложение заметки на Flutter

## Задание

1. Создать приложение на Flutter, которое будет представлять собой заметки. 
2. Пользователь сможет вводить текст в поле, и при нажатии на кнопку "Сохранить" введённый текст будет отображаться ниже в виде списка.
3. Добавьте возможность удаления заметок из списка.
4. Добавьте возможность редактирования существующих заметок.

1. Создайте простое приложение на Flutter, которое сохраняет и отображает список любимых фильмов. 
2. Используйте Hive/SQLite для хранения данных о фильмах. 
3. Реализуйте возможность добавления и удаления фильмов из списка. 
4. Добавьте возможность сохранять дополнительные данные о каждом фильме (например, год выпуска и жанр).
5. Реализуйте функциональность редактирования информации о фильме. 
6. Реализуйте добавление картинки к фильмам и их отображение 
7. Расширьте приложение с настройками, добавив переключатели для выбора темы приложения (светлая/темная).
8. Сохраните выбранную тему в Shared Preferences и применяйте её при запуске приложения.
