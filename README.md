# Практическая работа 3 - Работа с файлами

## Описание

Как сделать чтобы работало?
- Скачайте расширение Dart в VSC
- Запустите код через терминал командой `durt run`

## Цель

1. Дополнить своё приложение использованием асинхронных вызовов: использовать Future и Stream, реализовать изоляты в приложении
2. Работа с файлами: создавать при запуске игры файлики с данными игроков (имя, сколько игр было сыграно, кол-во побед и поражений), если файлик игрока уже существует, то новый создаваться не должен, должны изменяться данные существующего.
3. Добавить файл с данными о текущей игре: "игрок1"(имя игрока) : 5 попаданий, 5 промахов, 3 корабля целы, 1 подбит, 2 потоплено и т.д.
4. После окончания игры файлик должен либо очищаться (быть абсолютно пустым) либо удаляться.
5. Реализовать простенький лог файл, в нем: действия игроков, например Игрок1 сделал ход на 1А, результат : подбит/не подбит.
6. В случае ошибки работы программы, должен выводиться текст ошибки, например: Игрок1 пытался сделать ход на поле 1А, он уже ходил на него ранее, вызвана ошибка: "Ошибка, вы уже ходили на данное поле" или Игрок1 пытался поставить корабль на уже занятое поле при расстановке кораблей, вызвана ошибка: "Ошибка, поле уже занято кораблем"
